# Validacin cruzada
resultados = {}
for nombre, modelo in modelos.items():
    r2 = cross_val_score(modelo, X, y, cv=5, scoring='r2').mean()
    resultados[nombre] = r2

# Encontrar el mejor modelo
mejor_modelo_nombre = max(resultados, key=resultados.get)
mejor_modelo = modelos[mejor_modelo_nombre]

Error Cuadrtico Medio (MSE): Un MSE de 125606.54 indica que, en promedio, las predicciones del modelo tienen un error cuadrtico bastante alto. Esto sugiere que las predicciones no son muy precisas.

Coeficiente de Determinacin (R): Un R de 0.698 Este valor indica que el modelo de rbol de decisin explica aproximadamente el 69.8% de la variabilidad en la renta de bicicletas. Esto sugiere que el modelo tiene un rendimiento razonablemente bueno, pero hay espacio para mejoras.

La hora del da, la temperatura, la humedad y otros factores meteorolgicos (como la lluvia y la nieve) son determinantes importantes en la renta de bicicletas. El modelo identifica estos factores como los ms influyentes. Impacto de las Condiciones Meteorolgicas: Las condiciones meteorolgicas adversas, como altas temperaturas, lluvia y nieve, tienden a reducir la demanda de bicicletas.
